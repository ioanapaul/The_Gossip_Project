# Gossip Project
Our famous Gossip Project to transcribe in Rails. And as I explained to you, the first step is modeling its database!

The user journey is as follows: on this awesome social network, a user will sign up, provide their first and last name, email, and age, and then specify their city by searching with a postal code.

They will then have all the features that will make this app a future unicorn in Bitchin'Tech:

Users can create gossip: "apparently John is single hihi"
Users, when creating gossip, can add one or more tags to the gossip: #romance
Users can comment on gossip: "ahiii I didn't know lol ptdr 💁‍♂️"
And since your app must meet industry standards, we will make sure it is also possible to comment on comments.
Users can like gossip.
Users can contact their favorite gossips via private message to get world exclusives.
The user will be able to search for gossip by city, by users, by date (most recent or oldest), by number of likes, by tags, to find the juiciest gossip
